# RTCMeasurement

This repository contains datasets collected for measuring real-time communication (RTC) performance over Starlink and related network environments. The measurements are organized into two main categories: in-flight measurements and stationary node measurements.

## Dataset Overview

The dataset includes RTC measurement logs from two different experimental settings:

1. **In-flight measurements**
2. **Stationary node measurements**

The goal of these measurements is to analyze the behavior of real-time communication applications over Starlink.
